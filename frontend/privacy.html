개인정보처리방침

앱: 장보기 친구 (오늘의 마트 가이드) · 시행일 2026-05-31

1. 수집하는 개인정보

장보기 친구는 어떠한 개인정보도 수집·저장·전송하지 않습니다. 회원가입·로그인이 없으며, 이름·이메일·전화번호·위치 등 개인을 식별할 수 있는 정보를 요구하지 않습니다.

2. 기기 내 저장 정보

사용자가 선택한 재료 목록과 일부 조회 결과는 사용자 기기 내(브라우저 localStorage)에만 저장되어 화면 표시에 사용됩니다. 외부로 전송되지 않으며, 앱(브라우저) 데이터를 지우면 함께 삭제됩니다.

3. 외부 데이터 조회

식재료 가격·레시피 정보는 다음 공개 데이터 출처에서 조회하며, 이 과정에서 개인정보를 전송하지 않습니다(재료 이름 등 비개인정보만 사용).

  • KAMIS 농산물유통정보 (한국농수산식품유통공사) — 가격 시세
  • 네이버 검색/쇼핑 API — 레시피·온라인 최저가
  • 유튜브 데이터 API — 레시피 영상
  • 만개의레시피 — 레시피 결과

4. 광고·추적

앱 내 광고가 없으며, 사용자를 추적하는 분석·광고 도구를 사용하지 않습니다.

5. 제3자 제공

수집하는 개인정보가 없으므로 제3자에게 제공하는 개인정보도 없습니다.

6. 아동

본 앱은 개인정보를 수집하지 않으므로 모든 연령이 안전하게 이용할 수 있습니다.

7. 문의

개인정보 관련 문의: CONTACT_EMAIL (앱 등록 시 연락 가능한 이메일로 교체)

본 방침은 변경될 수 있으며, 변경 시 본 페이지에 게시합니다.
